Background

This application will analyze the thermodynamics of Carbon Monoxide with Oxygen flame.

\[\begin{aligned} \nu_{CO_2} CO + \nu_{O_2} O_2 \rightarrow \nu_{CO_2} CO_2 \end{aligned}\]

The heating value of this reaction is kJ/kmol or kJ/kg. Since no H2O is produced, the lower and higher heating value are the same.

Depending on the Fuel Equivalence Ratio (φ), the reaction can either be stoichiometric, fuel lean or fuel rich.

\[\begin{aligned} \text{Stoichiometric: } CO + 0.5 O_2 \rightarrow CO_2 \end{aligned}\]

\[\begin{aligned} \text{Fuel Lean: } CO + \left(\frac{1}{FAR}\right) O_2 \rightarrow CO_2 + \left(\frac{1}{FAR} - \frac{1}{2}\right) O_2 \end{aligned}\]

\[\begin{aligned} \text{Fuel Rich: } CO + \left(\frac{1}{FAR}\right) O_2 \rightarrow \left(\frac{2}{FAR}\right) CO_2 + \left(1 - \frac{2}{FAR}\right) CO \end{aligned}\]

This application calculates the adiabatic flame temperature for different Fuel Equivalence Ratios (φ). It does so using the 1st Law of Thermodynamics and by balancing the energy equation.

\[\begin{aligned} E_{in} - E_{out} = 0 \end{aligned}\]

\[\begin{aligned} \sum_{in} n_{in} h_{in} - \sum_{out} n_{out} h_{out} = 0 \end{aligned}\]

\[\begin{aligned} \sum_{in} n_{in} \left[ h_f^o + h(T_{reactants}) - h(T_{ref}) \right] - \sum_{out} n_{out} \left[ h_f^o + h(T_{products}) - h(T_{ref}) \right] = 0 \end{aligned}\]

By iterating over different values for Tproducts (bisection for this application), it is possible to solve the above equation.
